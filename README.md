# Module 3 Assignment – Responsive Web Page (Food, LLC)

## Project Description
This project is a responsive web page created using HTML, CSS, JavaScript, and Bootstrap. The webpage represents a restaurant menu for "Food, LLC" and adjusts its layout for desktop, tablet, and mobile screen sizes.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 3
- GitHub Pages (for hosting)

## Features
- Responsive layout using Bootstrap grid system
- Navigation bar with company name
- Mobile dropdown menu (Chicken, Beef, Sushi)
- Three menu sections with content
- Back to Top links
- Scroll to Top button
- Hover animation effects
- Dynamic content using JavaScript

## Responsive Layout
| Device | Layout |
|--------|--------|
| Desktop | 3 columns |
| Tablet | 2 columns on top, 1 column below |
| Mobile | 1 column stacked with dropdown menu |

## How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in a web browser
